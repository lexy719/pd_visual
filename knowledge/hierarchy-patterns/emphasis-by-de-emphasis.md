## emphasis-by-de-emphasis
tags: hierarchy, emphasis, contrast, typography, color, layout

To make one thing stand out, make everything else quieter — don't make it louder.

**Problem it solves** — the instinct is to emphasise by adding: bigger, bolder, brighter. When everything is emphasised nothing is, and the page becomes an argument between elements.
**When to use it** — whenever a section has a clear primary element and the supporting content is competing with it.
**Structure** — Choose the one element that must be read first → hold it at full contrast → step every neighbour down (lighter weight, muted foreground, smaller size) until it recedes.
**Why it works** — hierarchy is relative, not absolute. Contrast is a fixed budget: the cheapest way to buy emphasis is to stop spending it elsewhere, which also lowers the page's overall visual noise.
**Expected visual outcome** — the page looks quieter overall, yet the primary element reads as more prominent than it did when everything was bold.
**Examples** — a hero where the supporting line drops to muted foreground; a card grid where only the featured card keeps full contrast.
