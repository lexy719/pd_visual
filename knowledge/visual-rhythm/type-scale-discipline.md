## type-scale-discipline
tags: typography, scale, hierarchy, rhythm

Pick one ratio, derive every size from it, and use as few steps as the page can survive.

**Problem it solves** — ad-hoc sizes (17px here, 23px there) read as sloppy without anyone being able to say why. Sizes that are close but not equal look like mistakes rather than decisions.
**When to use it** — every page. The ratio is a brand decision: tighter (1.2) for dense/technical, wider (1.333-1.5) for editorial and premium.
**Structure** — Choose a ratio → generate the scale from a base body size → use 4-6 steps total across the whole page → if two elements need different emphasis, change weight or colour before adding a seventh size.
**Why it works** — a shared ratio makes every size look intentional even to a reader who never notices why. Few steps force real hierarchy decisions; many steps let everything be slightly special, which is the same as nothing being special.
**Expected visual outcome** — the page reads as typographically composed; size differences are decisive rather than marginal, and no two elements sit at nearly-but-not-quite the same size.
**Examples** — a premium editorial page on a 1.5 ratio with five sizes total, distinguishing everything else through weight and muted colour.
